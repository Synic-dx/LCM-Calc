# LCM-Calc
A Package to calculate the Least-Common-Multiple of any two numbers.

### Install:

``` bash
npm install lcm-calc
```

### Usage:

``` JavaScript
import lcmCalc from 'lcm-calc';

console.log(lcmCalc(12, 15));
```

This will log the LCM- 60 in this case.
